# Design Requirements

## üé® Signboard Design Principles

### Typography
- **Large, Readable Text:** Minimum 48px for body text, 72px+ for headings
- **High Contrast:** Dark text on light background or vice versa
- **Font Choice:** Sans-serif, clear and readable from distance
- **Line Height:** Generous spacing (1.5-2x font size)

### Color
- **High Contrast:** WCAG AA minimum (4.5:1 for normal text, 3:1 for large text)
- **Background:** Solid colors, avoid patterns
- **Text:** Dark on light or light on dark (avoid similar colors)

### Layout
- **Full-Screen:** Signs use full viewport
- **Padding:** Generous padding (5-10% of screen)
- **Grid:** Simple, clear grid for menu items
- **Spacing:** Large spacing between elements

## üìê Display Considerations

### Target Resolutions
- **Common:** 1920x1080 (Full HD), 3840x2160 (4K)
- **Aspect Ratio:** 16:9 (typical), 4:3 (some displays)
- **Orientation:** Landscape (primary)

### Image Requirements
- **Resolution:** Match or exceed display resolution
- **Format:** WebP (preferred), JPG, PNG
- **Optimization:** High quality, optimized file size
- **Aspect Ratio:** Match display or use appropriate ratios

## üé≠ Component Styling

### Menu Items
- Large, clear text
- Prominent price display
- Optional images (high quality)
- Clear separation between items

### Text Displays
- Large font sizes
- High contrast
- Generous line spacing
- Easy to read from distance

### Images
- Full-width or contained with padding
- High resolution
- Clear, professional images
- Optional captions (readable size)
